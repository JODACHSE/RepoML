Instalación Flask

Idealmente lo mejor es crear un entorno virtual de python asi:

  • mkdir myproject
  • cd myproject
  • python -3 -m venv .venv
  • .venv\Scripts\activate

Una vez activado instale flask:

  • python -m pip install flask

Para validar que quedo instalada ejecutar:

  • python -m flask --version

Para levantar el servidor con flask:

  • python -m flask run

Flask instala automáticamente algunas dependencias como

  • Werkzeug implementa WSGI, la interfaz estándar de Python entre aplicaciones y servidores.
  • Jinja es un lenguaje de plantilla que representa las páginas que atiende su aplicación.
  • MarkupSafe viene con Jinja. Escapa de la entrada no confiable al representar plantillas para evitar ataques de inyección.
  • Itsdangerous firma datos de forma segura para garantizar su integridad. Esto se usa para proteger la cookie de sesión de Flask.
  • Click es un marco para escribir aplicaciones de línea de comandos. Proporciona el comando Flask y permite agregar comandos de administración personalizados.
  • Blinker proporciona soporte para señales.

Curiosidades

  • El nombre "Flask" es una referencia al framework anterior llamado Bottle.
  • Flask es conocido por su enfoque minimalista, pero es altamente extensible gracias a su comunidad activa.
  • Es ideal para proyectos pequeños y prototipos, pero también puede manejar aplicaciones complejas con extensiones.
